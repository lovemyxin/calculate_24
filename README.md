使用Godot4.5长期稳定版引擎开发的H5在线四则运算求解二十四点纸牌游戏。
实现思路：
1、四张牌有24种排列组合，这里去重，如四张牌分别为【2，4，10，10】时，其中可能会重现【2，4，10，10】，【4，2，10，10】等等，需要去掉。
2、确定四张牌的放置顺序之后，还需要考虑运算顺序，一共有5种。
     ((a b) c) d
     ((a b) (c d))
     a ((b c) d)
     a (b (c d))
     (a (b c))d
3、每种顺序种遍历填入+、-、*、/，使用Godot自带的Expression类对字符串进行计算，可以运算且结果与24误差小于10的负6此方时，说明此式子有用。
4、注意在逻辑内四张牌的类型为float，但是显示给用户看到的是int
